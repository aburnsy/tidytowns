# Pre-Adjudication Photography Shot List (PRIVATE)

Use this checklist to ensure photographic coverage across all 8 categories and all seasons.
The adjudicator visit typically falls in July. Aim to have a full seasonal library by June.

---

## Spring (March - April)

Spring photography captures early colour and demonstrates year-round effort.

### Community
- [ ] Community clean-up event in progress (wide shot showing participant numbers)
- [ ] Volunteers with litter bags on a named street
- [ ] School group participating in a Tidy Towns activity

### Streetscape
- [ ] Main street before hanging baskets are installed (shows base standard)
- [ ] Window boxes with spring bulbs (daffodils, tulips)
- [ ] Bollards at the monument — current condition (for before/after record)
- [ ] Derelict property frontage — current condition (for before/after record)

### Green Spaces
- [ ] Spring bulbs in all named planting beds
- [ ] Sensory garden — overall view and close-up of pathway condition
- [ ] Church grounds bed — current condition
- [ ] Glen Carraig entrance area — current condition

### Nature & Biodiversity
- [ ] Bog walk entrance — signage and path condition
- [ ] Bog walk path — wide shot showing character of the route
- [ ] Any bird boxes installed — with location context
- [ ] Hedgerow on approach roads — bare or budding

### Sustainability
- [ ] Water butts in place (if installed) — show connection to guttering or barrel
- [ ] Compost bin (if set up) — show contents and location

### Tidiness
- [ ] Mutt mitt dispenser at monument — full and functional
- [ ] Litter bin condition — all bins on main street

### Residential
- [ ] Noel Hayes Park — entrance and green strips
- [ ] Glen Carraig — entrance and any wildflower area
- [ ] Any notable front garden in early spring colour

### Approach Roads
- [ ] Each of the four main approach roads from the village boundary inward
- [ ] Verge condition — note any unmown strips

---

## Summer (May - July)

Peak season. Must have comprehensive coverage before the adjudicator visit (usually July).

### Community
- [ ] Summer event or festival — wide crowd shot
- [ ] Garden competition prize-giving (if held)
- [ ] Business sponsorship — photograph with named business owner
- [ ] Any inclusivity event in progress

### Streetscape
- [ ] Main street in full hanging basket season — wide panoramic
- [ ] Close-up of best window box arrangement
- [ ] Close-up of best hanging basket
- [ ] Bollards after repainting (if done)
- [ ] Forge window (existing or new)
- [ ] Any derelict property after improvement (if done)
- [ ] All planters and tubs in peak condition
- [ ] Village signage — condition and surroundings

### Green Spaces
- [ ] Every named planting bed in peak bloom (one photo each)
- [ ] Sensory garden — pathway clear and beds full
- [ ] Wildflower bed at church (if planted)
- [ ] Pollinator plants in close-up showing bees/butterflies if possible
- [ ] All-Ireland Pollinator Plan species — close-up with identification

### Nature & Biodiversity
- [ ] Bog walk — 3 or more shots showing different sections
- [ ] Bog walk — any notable flora or fauna spotted
- [ ] Any wildlife camera footage stills (if available)
- [ ] Tree planting area — trees with labels
- [ ] Graveyard extension — current state and any biodiversity planting
- [ ] Log pile or wildlife habitat feature — with context
- [ ] Any species sighting (butterfly, bird, frog) — with location note

### Sustainability
- [ ] Water butts — in active use
- [ ] Compost bin — being used
- [ ] Litter segregation in action (if implemented)
- [ ] Any eco weed management in progress (hand-weeding, vinegar spray)
- [ ] Any energy-saving feature (solar light, LED fitting)

### Tidiness
- [ ] Litter patrol in progress — volunteers on a named route
- [ ] Bags of litter collected — weigh and photograph
- [ ] All public bins — clean and not overflowing
- [ ] Mutt mitt dispenser — stocked
- [ ] Any graffiti removal — before/after

### Residential
- [ ] Each estate — entrance sign and general view
- [ ] Noel Hayes Park — bedding strips and wall climbers (if planted)
- [ ] Glen Carraig — wildflower area at entrance (if planted)
- [ ] Best three front gardens — with house number or owner permission
- [ ] Garden competition winner's garden — with winner if they consent

### Approach Roads
- [ ] Each approach road — from boundary to village (4 roads, 2 shots each)
- [ ] Best wildflower verge strip — close-up of species
- [ ] Welcome signage on each approach — condition and surroundings
- [ ] Any gateway planting in peak condition

### Nature & Biodiversity

### Sustainability
- [ ] Compost bin with autumn leaves being added
- [ ] Seed saving activity (if organised)

### Residential
- [ ] Any estate with autumn planting interest

- [ ] Christmas lights or seasonal decoration — community effort

### Streetscape
- [ ] Main street with winter interest planting (evergreens, winter pansies)
- [ ] Any festive planters or window boxes

### Green Spaces
- [ ] Sensory garden — evergreen structure visible
- [ ] Any winter-flowering plants

### Nature & Biodiversity
- [ ] Bog walk in winter — shows accessibility year-round
- [ ] Birds using feeders (if installed)
- [ ] Hedgerows providing winter habitat

### Tidiness
- [ ] Clean streets after a winter storm — shows maintenance commitment

## Photography Standards

- **Minimum resolution:** 2 megapixels for application attachments
- **Orientation:** Landscape format preferred for wide shots; portrait acceptable for close-ups
- **Metadata:** Note the date, location, and photographer for every photo
- **Naming convention:** `YYYY-MM-DD_Category_Subject.jpg` (e.g., `2026-06-15_Streetscape_MainStreet.jpg`)
- **Storage:** Back up all photos to the shared committee folder immediately after capture
- **Captions:** Write a one-line caption for every photo at the time of taking, while context is fresh
