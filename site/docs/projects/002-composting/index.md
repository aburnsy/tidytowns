# Start composting organic waste for reuse in beds

## Description

Set up a composting station for organic waste from village maintenance. Compost reused in flower beds and planters, closing the loop on green waste.

The 2025 adjudicator recommended introducing composting as part of a broader push toward sustainable waste management and circular economy principles.
