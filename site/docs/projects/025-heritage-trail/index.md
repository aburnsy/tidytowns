## Description

A formal heritage trail connecting the Norman castle, the old forge, the church, and the bog walk entrance would create a coherent visitor experience and showcase the village's rich history. A central map and wayfinding signs tie the individual sites into a unified trail. The exemplar village of Emly was praised by adjudicators for developing exactly this type of heritage trail, complete with a central interpretive map.
