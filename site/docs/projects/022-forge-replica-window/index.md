# Install replica window in the old forge

## Description

The old forge is a heritage building with significant character, but its empty window opening currently detracts from the streetscape. Installing a sympathetically designed replica window, guided by the County Heritage Officer, would restore the building's visual integrity and contribute to the overall quality of the village streetscape.

## What the adjudicator said

The 2025 adjudicator recommended installing a replica window in the old forge and advised engaging the County Heritage Officer for guidance on appropriate design and materials, as well as potential funding sources.
