# Improve application writing across all categories

## Description

High-performing TidyTowns villages consistently invest time in crafting clear, evidence-rich application text that tells a compelling story. Analysis of exemplar villages Emly and Silvermines shows that the quality of the written application can make a significant difference to how adjudicators assess a village's work. This project focuses on improving the structure, evidence, and language of our annual submission.
