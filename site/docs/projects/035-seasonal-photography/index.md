# Seasonal photography program across all 4 seasons

## Description

High-quality photographs taken across all four seasons allow us to demonstrate the full scope of the village's TidyTowns effort in a compelling, visual way. Both exemplar villages Emly and Silvermines were praised by adjudicators for their photographic documentation. A structured program to capture the village in spring, summer, autumn, and winter costs nothing and significantly enriches the application.
