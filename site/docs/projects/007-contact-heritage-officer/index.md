# Contact Heritage Officer about forge window restoration

## Description

The old forge is a distinctive heritage building in the village whose window opening currently detracts from the streetscape. The County Heritage Officer can advise on appropriate materials, design, and potential funding to install a replica window that restores the building's character at no or low cost to the community.

## What the adjudicator said

The 2025 adjudicator recommended contacting the County Heritage Officer for guidance on both the funding options and the correct methodology for installing a replica window in the forge.
